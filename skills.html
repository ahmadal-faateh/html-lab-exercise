Glossary

HTML
The standard markup language for creating web pages.
Semantic Element
An element that clearly describes its meaning to both the browser and the developer.
Attribute
Additional information provided about an HTML element, placed in the opening tag.
Hyperlink
Hyperlinks (or just links) take the user to another webpage when they click on it. The most common attribute used with links is href, which tells the browser where the link goes.
